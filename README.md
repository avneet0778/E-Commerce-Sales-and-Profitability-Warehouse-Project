# E-Commerce Sales and Profitability Analysis

This project analyzes e-commerce data using SQL to extract insights on revenue, customers, delivery performance, and sales trends.

## Contents:
- dataset/: contains dataset used for the analysis
- queries/: contains SQL queries used for analysis
- outputs/: contains key results generated from queries


## Key Insights:
- Monthly revenue trends identified
- Top product categories analyzed
- Customer behavior (repeat vs one-time) studied
- Delivery performance impact evaluated
